# hype-audio-app

Standalone motivation-audio app — upload short hype clips (Goggins/Dorian-style),
play them back by mentality/moment, filtered by pillar (`iron`/`faith`). Shares
`hype-audio.js` and `sync.js` byte-identical with the `row` repo (Row's `gym.html`
mini-player uses the same two files) — keep both copies in sync when either changes.

Live: https://hype-audio-app.vercel.app
